# wordle-solver

Dictionary data 
    words (102,401) from Ubuntu /usr/share/dict/words
    words_scrabble (172,820 words) from https://raw.githubusercontent.com/zeisler/scrabble/master/db/dictionary.csv

To run the program
    # bash wordle.bash
